# Projeto_AED

para instalar a lib:

para linux:

sudo apt-get update
sudo apt-get install libsdl2-dev

para macos:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"


brew update
brew install sdl2

para windons 

para buildar o jogo:
gcc main.c -o main -lSDL2

para executar:
./main
